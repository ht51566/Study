
OOP: Object Oriented Programming
 - 클래스로 찍어낸 객체들 간의 거래로 문제를 해결하도록 프로그래밍
 
자바의 4대 특징
1. 추상화: 대상의 특징(필요한 데이터)을 뽑아서 클래스 만들고 객체 생성
2. 캡슐화: 하나의 클래스가 가진 데이터를 외부에서 직접 접근할 수 없도록 막는다 - 객체들간의 간섭, 충돌을 막음(encapsulation - private)
3. 상속 : 클래스의 재사용 및 확장 - 기존의 데이터나 프로그램에 새로운 데이터나 기능을 쉽게 추가 가능
4. 다형성: 하나의 클래스를 상속받은 자식들이 부모 타입을 통해 각자의 기능을 수행 가능 - Overriding(고쳐 씀)
